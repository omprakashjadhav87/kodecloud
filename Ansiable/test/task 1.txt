1) Generate SSH Key Pair on jump host: Run the following command to generate a new SSH key pair:

ssh-keygen -t rsa -b 2048 -f ~/.ssh/id_rsa

2) Copy Public Key to Remote Server: Use the ssh-copy-id command to copy the public key to stapp02( Server name or ip address:

ssh-copy-id steve@172.16.238.11

3) Update Ansible Inventory: Ensure that the inventory file has the correct user and host details. It should look like this:

[app_servers]
stapp02 ansible_host=172.16.238.11 ansible_user=steve 

4) ansible -i inventory stapp02 -m ping  ( this should be successfully ping ignore warning only)

5 ) create ansible playbook .( cd /home/thor/ansible ) 

---
- name: Create an empty file on App Server 2
  hosts: stapp02
  tasks:
    - name: Create an empty file
      file:
        path: /tmp/file.txt
        state: touch


6 ) Run the Playbook: To run the playbook, navigate to the directory where your playbook is located and use the ansible-playbook command. For example:

cd /home/thor/ansible
ansible-playbook -i inventory playbook.yml
